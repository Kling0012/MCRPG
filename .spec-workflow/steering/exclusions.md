# 意図的に実装しない機能

## 概要

このドキュメントは、RPGPluginで**意図的に実装しない**機能を明記します。
これらは将来的な機能追加の候補ではなく、プロダクトの方向性として**実装しない**ことが決定されている機能です。

---

## 1. モブ討伐時のゴールドドロップ

### ステータス
- **実装しない**

### 理由
- 経済システムはプレイヤー間の取り引き（オークション、トレード等）を中心に設計されている
- ゴールドは他の手段（クエスト、デイリーボーナス等）で獲得可能

### 補足
- `DamageTracker`クラスが実装されており、トドメを与えたプレイヤー（last hitter）を特定可能
- クエストシステム等で`DamageTracker.getKiller()`を使用して報酬を付与することは可能

### 参照
- `CurrencyListener.java:56-65` - ゴールドドロップは実装していない
- `DamageTracker.java` - トドメを与えたプレイヤーの特定に使用可能

---

## 2. パッシブスキルのポーション効果付与

### ステータス
- **実装しない**

### 理由
- パッシブスキルはステータス補正（STR/INT/SPI/VIT/DEX）で十分に機能する
- ポーション効果（速度、再生等）はアクティブスキルやアイテムで提供可能
- 設定の複雑化を避けるため、パッシブスキルはステータス補正に特化

### 参照
- `PassiveSkillExecutor.java:123-125` - ポーション効果は実装しない

## 4. 効果のないスキル（発動のみ）

### ステータス
- **許可する** - 効果がないスキルの発動を許可

### 概要
ダメージや効果が設定されていないアクティブスキルでも発動を許可します。
これらのスキルは以下の動作をします：
- **コスト消費**: MP/HPを消費する（設定されている場合）
- **クールダウン**: クールダウンを設定する
- **ターゲット選択**: 周囲のエンティティをターゲットとして取得
- **メッセージ表示**: 発動成功メッセージとターゲット数を表示

### 用途
- **プレースホルダー**: 開発中のスキルの仮実装
- **他システム連携**: クエストシステム等から発火されるスキル
- **コスメティック**: エフェクトや音のみのスキル

### メッセージ例
```
# ダメージありスキル
スキルを発動しました: ファイアボール Lv.3 (MP消費: 50) [対象: 2]

# 効果なしスキル（コストあり）
スキルを発動しました: チャント Lv.1 (MP消費: 20) [対象: 3]

# 効果なしスキル（コストなし）
スキルを発動しました: 挑発 Lv.5 [対象: 1]
```

### 参照
- `ActiveSkillExecutor.java:19-45` - JavaDocに説明あり
- `ActiveSkillExecutor.java:66-153` - execute()メソッド実装

## 5. スキルターゲット設定

### ステータス
- **実装済み**

### 概要
YAMLでターゲットの種類と対象エンティティを設定できます。

### ターゲットタイプ
| タイプ | 説明 |
|--------|------|
| `self` | 自分自身のみ |
| `self_plus_one` | 自分と最も近いターゲット一人 |
| `nearest_hostile` | 最も近いMob |
| `nearest_player` | 最も近いプレイヤー |
| `nearest_entity` | 最も近いエンティティ |
| `area_self` | 自分を含む範囲内の全て |
| `area_others` | 自分を含まない範囲内の全て |

### エンティティフィルタ
| フィルタ | 説明 |
|---------|------|
| `player_only` | プレイヤーのみ |
| `mob_only` | Mobのみ |
| `all` | 全て（デフォルト） |

### YAML設定例
```yaml
# 自分と最も近いプレイヤー一人
target:
  type: self_plus_one
  entity_filter: player_only

# 範囲内のMob最大5体
target:
  type: area_others
  entity_filter: mob_only
  max_targets: 5
  range: 10.0
```

### 参照
- `TargetType.java` - ターゲットタイプ定義
- `EntityTypeFilter.java` - エンティティフィルタ定義
- `SkillTarget.java` - ターゲット設定クラス
- `TargetSelector.java` - ターゲット選択ロジック

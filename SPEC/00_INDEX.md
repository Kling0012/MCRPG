## 核心機能

1. **ダメージシステム**: 全イベントキャッチ、ステータス倍率、ダメージカット
2. **ステータスシステム**: STR/INT/SPI/VIT/DEX、レベルアップ時「自動+2 / 手動配分3ポイント」
3. **クラスシステム**: 4初期クラス、ランク6までクラスアップ、直線/分岐両対応
4. **スキルシステム**: 共通スキルプール、クラスで選択、数式ベースダメージ計算
5. **ストレージ**: SQLite + 3層キャッシュ（50-150人対応）
6. **MythicMobs連携**: ドロップ管理（倒した人のみ）
7. **外部API**: PlaceholderAPI、Java API
8. **バニラ経験値**: 減衰設定、各クラスで上限

## 初期クラス

- **戦士**: 剣と盾で前線を守る近接クラス
- **大盾使い**: 盾特化の防御クラス
- **魔法使い**: 魔法で攻撃する遠距離クラス
- **弓使い**: 弓で攻撃する遠距離クラス
